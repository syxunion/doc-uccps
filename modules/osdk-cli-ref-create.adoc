[id="osdk-cli-ref-create_{context}"]
= create

`operator-sdk create` 命令用于创建或 _scaffold_  Kubernetes API。

[id="osdk-cli-ref-create-api_{context}"]
== api

`create api` 子命令构建 Kubernetes API。子命令必须在 `init` 命令初始化的项目中运行。
