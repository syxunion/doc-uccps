= 快速入门突出显示 markdown 参考

高亮显示（或提示）功能可让快速入门包含可突出显示并模拟 web 控制台组件的链接。

markdown 语法包含：

* 括起的链接文本
* `highlight` 关键字，后跟您要动画的元素的 ID
