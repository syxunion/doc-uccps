= 使用 web 控制台删除项目

您可以通过 统信容器云管理平台 Web 控制台删除一个项目。

[注意]
====
如果您没有删除项目的权限，*Delete Project* 选项将无法使用。
====

.流程

. 导航到 *主页* -> *项目*.

. 找到您要从项目列表中删除的项目。

. 在项目列表的最右侧，从选项菜单中 {kebab} 选择 *删除项目*。

. 打开 *删除项目* 界面时，在字段中输入要删除的项目名称。

. 点击 *删除*。
