= 配置警报通知
include::_attributes/common-attributes.adoc[]
:context: configuring-alert-notifications

toc::[]

在 统信容器云管理平台 中，当警报规则中定义的条件满足时会触发警报。警报提供一条通知，表示集群中存在一组情况。默认情况下，触发的警报可在 统信容器云管理平台 Web 控制台中的 Alerting UI 中查看。安装后，您可以配置 统信容器云管理平台，将警报通知发送到外部系统。
