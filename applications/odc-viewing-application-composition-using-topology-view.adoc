= 使用拓扑视图查看应用程序组成情况
include::_attributes/common-attributes.adoc[]
:context: viewing-application-composition-using-topology-view

toc::[]

Web 控制台的 *开发者* 视角中有一个 *拓扑* 视图，它以可视化方式展示项目中的所有应用程序、它们的构建状态，以及关联的组件和服务。

== 先决条件
要在 *拓扑* 视图中查看应用程序并与之交互，请确保：

* 已登陆到 web 控制台。
* 处于 *开发者* 视角。
* 在项目中拥有适当的角色和权限，可在 统信容器云管理平台 中创建应用程序和其他工作负载。
* 您已使用 *开发者* 视角在 统信容器云管理平台 上创建并部署了应用程序。
