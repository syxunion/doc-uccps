= 访问 Web 控制台

统信容器云管理平台 Web控制台是可从Web浏览器访问的用户界面。开发人员可以使用Web控制台来直观地浏览并管理项目的内容。

== 先决条件

* 必须启用JavaScript才能使用Web控制台。为获得最佳体验，请使用支持WebSockets的Web浏览器。
* 在为集群创建支持基础结构之前，请参阅统信容器云管理平台 4.x Tested Integrations页。
